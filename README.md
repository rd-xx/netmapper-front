# netMAPPER - Frontend

Projet créé dans le cadre du cours de **Node.js**.

## Développement

### Prérequis

- [Node.js](https://nodejs.org/en/download/current)

### Installation

1. Clonez le dépôt.
2. Installez les dépendances avec `npm install`.
3. Lancez le serveur de développement avec `npm run dev`.
